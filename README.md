# Timer

Минималистичный desktop-таймер для Windows с треем, статистикой времени и быстрым ручным вводом.

Сделан для простого учёта времени без перегруженного интерфейса, аккаунтов, облаков и прочего цифрового мусора цивилизации.

---

## Возможности

* ▶ Старт / пауза таймера
* 🔄 Быстрый сброс с возможностью восстановления
* ⏱ Сохранение сессий
* 📊 Статистика:

  * за сегодня
  * за неделю
  * за всё время
* ✍ Ручной ввод времени
* 🖥 Сворачивание в системный трей
* 🔔 Tooltip с текущим временем в трее
* 🚫 Только один экземпляр приложения
* 📦 Portable `.exe` сборка

## Управление

### Основные кнопки

| Кнопка | Действие                              |
| ------ | ------------------------------------- |
| ▶      | Старт / пауза                         |
| 🔄     | Сброс времени                         |
| ⏱      | Сохранить сессию / открыть статистику |

---

### Ручной ввод времени

Двойной клик по таймеру открывает поле ввода.

Поддерживаемые форматы:

```text
01:30.50
01:20:15
90m
1h30m
45s
```

---

## Статистика

После сохранения сессий кнопка ⏱ показывает:

* время за сегодня
* время за последние 7 дней
* общее время

Данные сохраняются в:

```text
laps_data.json
```

---

## Системный трей

Закрытие окна не завершает приложение.

Timer продолжает работать в системном трее.

Функции трея:

* Start/Pause
* Show
* Quit

## Почему этот таймер существует

Потому что большинство таймеров либо:

* выглядят как cockpit Boeing 737,
* требуют регистрацию,
* продают подписку,
* синхронизируются с “облаком” ради таймера на 15 минут.

Иногда нужен просто таймер.
