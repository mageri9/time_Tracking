# Timer Mimer — секундомер на Tkinter

Небольшое десктоп-приложение на Python/Tkinter:

- секундомер с паузой и сбросом;
- запись кругов с сохранением в `laps_data.json`;
- просмотр статистики по кругам;
- ручной ввод/редактирование времени прямо по двойному клику.

## Требования

- Python 3.10+ (подойдёт и более новый);
- дополнительных библиотек устанавливать не нужно — используется только стандартная библиотека.

## Установка и запуск

1. Клонируй или скачай репозиторий в папку, например:

   ```bash
   cd timer_mimer
   ```

2. Запуск приложения (есть несколько вариантов):

   - как модуль:

     ```bash
     python -m stopwatch
     ```

   - через скрипт `main.py`:

     ```bash
     python main.py
     ```

   - через скрипт `test.py` (оставлен для удобства разработки):

     ```bash
     python test.py
     ```

## Управление

- большая кнопка `▶ / ⏸` — старт/пауза;
- кнопка `🔄` — сброс времени (с возможностью отмены первого сброса);
- кнопка `⏱️` — запись круга / показ статистики.

### Закрытие программы

Если попытаться закрыть окно, пока секундомер идёт, появится диалог-подтверждение.
При выборе «Отмена» окно останется открытым, при «OK» приложение закроется.

### Ручной ввод времени

Двойной клик по табло времени открывает поле ввода. Поддерживаются форматы:

- классические:
  - `ЧЧ:ММ:СС` — например, `70:00:00` (70 часов);
  - `ММ:СС.сотые` — например, `05:30.25`;
- «человеческие» с буквами (латиница/кириллица):
  - `70h`, `70ч`;
  - `1h30m`, `1ч30м`;
  - `90m15s`, `90м15с`;
  - `30s`, `30с`;
  - пробелы допускаются: `1h 30m`, `1ч 30м` и т.п.;
- просто число:
  - `70` — считается как 70 часов.

При неверном формате покажется сообщение «Неверный формат времени» в информационной панели.

## Структура проекта

```text
timer_mimer/
  stopwatch/
    __init__.py      # экспорт основных классов
    __main__.py      # точка входа: python -m stopwatch
    models.py        # dataclass-состояние секундомера
    controllers.py   # бизнес-логика (start/stop/reset/lap, работа с laps_data.json)
    views.py         # Tkinter GUI, обёртка над контроллером
    utils.py         # parse_time, format_time и вспомогательные функции
  tests/
    test_utils.py    # примеры unit-тестов
  main.py            # запуск секундомера через stopwatch.__main__.main()
  test.py            # альтернативный запуск для разработки
  laps_data.json     # файл с кругами (создаётся автоматически)
```

## Сборка `.exe` для Windows

Для сборки одного `.exe` (без консольного окна) используется [PyInstaller](https://pyinstaller.org).

1. Установи PyInstaller (один раз):

   ```bash
   pip install pyinstaller
   ```

2. В корне проекта (`timer_mimer/`) выполни:

   ```bash
   build_exe.bat
   ```

   или руками:

   ```bash
   pyinstaller --noconfirm --noconsole --onefile --name TimerMimer main.py
   ```

3. Готовый файл появится в папке `dist`:

   ```text
   dist/TimerMimer.exe
   ```

Этот `.exe` можно положить на рабочий стол и запускать двойным кликом (Python на машине пользователя уже не обязателен).
